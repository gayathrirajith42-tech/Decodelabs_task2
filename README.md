# Exploratory Data Analysis (EDA)

## Project Overview

This project was completed as part of the DecodeLabs Data Analytics Internship. The objective of this project is to perform Exploratory Data Analysis (EDA) on a cleaned dataset to understand its structure, summarize key characteristics, and identify relationships between variables.

## Objective

The main objectives of this project are:

- Understand the dataset through statistical analysis
- Calculate descriptive statistics
- Determine mean, median, and count values
- Analyze relationships between variables
- Identify trends and patterns within the data
- Perform correlation analysis

## Tools and Technologies Used

- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset

The analysis was performed on the cleaned dataset generated during Project 1 (Data Cleaning and Preparation).

## Analysis Performed

### 1. Data Inspection

- Loaded the cleaned dataset
- Displayed the first five rows
- Examined dataset structure and data types

### 2. Descriptive Statistics

Generated summary statistics including:

- Count
- Mean
- Standard Deviation
- Minimum Value
- Maximum Value
- Quartiles

### 3. Mean Analysis

Calculated the average values of numerical columns to understand central tendencies.

### 4. Median Analysis

Calculated median values to identify the middle value of numerical distributions.

### 5. Count Analysis

Determined the number of non-null values in each column.

### 6. Correlation Analysis

Analyzed relationships between numerical variables using a correlation matrix.

### 7. Data Visualization

Created visual representations of the dataset using:

- Histograms
- Correlation Matrix Visualization

## Key Findings

- Numerical variables were summarized using descriptive statistics.
- Mean and median values provided insights into data distribution.
- Correlation analysis revealed relationships between variables.
- Visualizations helped identify patterns and trends within the dataset.
